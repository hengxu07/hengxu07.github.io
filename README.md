# hengxu07.github.io

Personal portfolio site for Heng Xu — built with [Astro](https://astro.build) and deployed to GitHub Pages.

## Projects

- **[Dokkai](https://dokkai-lilac.vercel.app/)** — AI-powered Japanese reading assistant. Upload any text or PDF and get vocabulary extraction, grammar analysis, JLPT level assessment, and auto-generated quizzes.
- **[Namicast](https://namicast.vercel.app/)** — AI-powered surf forecast app. Ask any question about any spot and get a personalized recommendation based on your board and skill level.
- **[Powderseek](https://powderseek.vercel.app/)** — AI-powered ski and snowboard trip planner. Tell it how many days you have and it recommends where to go based on live snow forecasts, travel time, skill level, and budget.

## Stack

Astro · TypeScript · GitHub Pages

## Commands

| Command             | Action                                     |
| :------------------ | :----------------------------------------- |
| `npm install`       | Install dependencies                       |
| `npm run dev`       | Start local dev server at `localhost:4321` |
| `npm run build`     | Build for production to `./dist/`          |
| `npm run preview`   | Preview production build locally           |

## Adding photos

Personal photos for the interactive identity pills live in `src/assets/`:

- `src/assets/cats/` — cat photos (🐱 Cat Lover pill)
- `src/assets/snowboarding/` — snowboarding photos (🏂 Snowboarder pill)
- `src/assets/surfing/` — surfing photos (🏄 Surfer pill)
- `src/assets/japan/` — Japan trip photos (🎌 Japan pill)

Drop any `.jpg`, `.jpeg`, `.png`, or `.webp` files in the relevant folder and they'll appear automatically on the next build.
